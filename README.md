### Цель работы 
Создать киент-серверное приложение для бронирования и покупки билетов в кино
### Функционал
- Вход в систему
- Выход из системы
- Регистрация 
- Поиск фильма по фильтрам
- Просмотр информации о фильме
- Бронирование фильмов
- Купить билет
